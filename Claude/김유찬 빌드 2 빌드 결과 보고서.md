## 테스트 결과

### 정상 동작 확인 항목

**Tribute System**

- 제단 목록 아이템 클릭 시 인벤토리에서 해당 아이템 하이라이트
- 아이템 제출 시 인벤토리 차감
- 아이템 제출 완료 시 체크 표시

**Craft System**

- 아이템 클릭 시 필요 재료 목록 표시
- 재료 충족 시 초록색, 부족 시 빨간색 표시
- 아이템 제작 시 인벤토리에 추가

**Storage System**

- 인벤토리 아이템 클릭으로 창고 이동
- 절반 이동 / 전체 이동 가능
- 플레이어 사망 후에도 창고 데이터 유지

**Animal AI**

- 토끼(초식동물): Idle → Roam → Flee 정상 동작
- 곰(육식동물): 특정 상황에서 Attack 상태 미진입 (버그)

**타일 기반 맵 생성**

- 7일차마다 맵·자원·동물 재생성 정상 동작

**사망 시스템**

- 사망 시 오두막에서 리스폰, 인벤토리 초기화
- 동물 공격에 의한 사망 적용
- 시간 초과 사망 적용

**Upgrade System**

- 업그레이드 항목 클릭 시 필요 재료 표시
- 업그레이드 후 다음 단계 재료로 업데이트
- 업그레이드 효과 미구현

**Equipment System**

- 인벤토리에서 장비 착용 가능
- 부위별 착용 가능한 아이템 구분
- 인벤토리 아이템 클릭 → 장착 / 제거 버튼
- 장착 아이템 클릭 → 해제 버튼
- 같은 부위에 장착 중인 아이템 있으면 기존 장비가 인벤토리로 반환

**Game Over System**

- 게임 오버 여부는 7일차 제단 충족 여부로만 결정
- 도중에 여러 번 사망해도 7일 안에 제출하면 클리어
- 제출 완료 시 다음 날로 진행
- 제출 미완료 시 게임 오버 화면 표시

### 미검증 / 미완성 항목

- 업그레이드 효과 (창고 용량 증가, 제작대 아이템 확장 등)
- 장비 착용 효과 (체력 증가, 공격력 증가)
- 용량 초과 시 아이템 소실 방지 처리
- 가마솥 요리 제작 기능
- 배고픔 시스템
