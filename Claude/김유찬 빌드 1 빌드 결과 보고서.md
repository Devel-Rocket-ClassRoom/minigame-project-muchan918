## 테스트 결과

### 정상 동작 확인 항목

- WASD 이동, 이동 방향 회전, 애니메이션 블렌딩
- 자원 채집 → 아이템 드롭 → 줍기 → 인벤토리 등록
- 낮/밤 사이클 흐름, 날짜 카운트 증가
- 취침 → 다음 날 아침, 체력 완전 회복
- 제단 인터랙션 → 패널 열기/닫기
- 7일차 아침 공물 충족 시 → 다음 사이클 진행
- 7일차 아침 공물 미달 시 → 게임오버 화면 표시

### 미검증 / 미완성 항목

- 공물 제출 창 UI (요구 목록, 보유량, 부분 제출)
- 제출 시 인벤토리 실제 차감
- 7일차 충족/미달 결과 화면 분기
- 핵심 루프 1회 완주 통합 테스트

---

## 발견된 문제점

- 플레이어가 움직일 때 특정 상황에서 Hp와 배고픔 슬라이더가 자동으로 움직임
- 배고픔을 조절하는 스크립트는 작성하지 않았는데 움직여서 문제
- 인벤토리에 같은 물건은 쌓여서 채우도록 구현하지 못함
- 맵 끝 처리 부족

---

## 총평

핵심 시스템(이동/채집/인벤토리/HP/낮밤/제단/7일차 평가)은 모두 구현 완료. 공물 제출 UI와 7일차 결과 화면이 빠져 있어 아직 완전한 루프 1회 완주는 불가한 상태. 2차 빌드에서 해당 항목 완성 후 통합 테스트 예정.
