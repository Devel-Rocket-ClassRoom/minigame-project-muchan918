# 7Days — 2차 빌드 노트

> 빌드 날짜: 2026-05-29
> 빌드 대상: Android
> Unity 버전: 6.3 / URP

---

## 구현된 기능

### Tribute System

- 아이템을 제단에 제출하는 기능
- 화면 기준 오두막 오른쪽 빨간색 포탈에 들어가면 상호작용 버튼 활성화
- 상호작용 버튼을 누르면 Tribute Panel 표시
- 인벤토리 아이템 제출 가능
- 인벤토리 아이템 최대 개수와 제단 목록 아이템 개수를 계산해 수량 자동 설정
- 슬롯 더블 클릭으로 수량 증가 / 플러스·마이너스 버튼으로 수량 조절
- 제단 목록 아이템 제출 시 체크 표시
- 제단 목록 아이템 클릭 시 인벤토리에서 해당 아이템 하이라이트

### Craft System

- 아이템 제작 시스템
- 화면 기준 오두막 왼쪽 노란색 포탈에 들어가면 상호작용 버튼 활성화
- 상호작용 버튼을 누르면 Craft Panel 표시
- 제작 가능한 아이템 목록 표시, 클릭 시 필요 재료 목록 표시
- 재료 충족 시 초록색, 부족 시 빨간색으로 표시
- 제작 버튼을 누르면 인벤토리에 아이템 추가

### Storage System

- 아이템 저장 창고 시스템
- 화면 기준 오두막 왼쪽 대각선 아래(제작대 아래) 보라색 포탈
- 상호작용 버튼을 누르면 Storage Panel 표시
- 인벤토리 ↔ 창고 간 아이템 이동 가능
- 절반 이동 / 전체 이동 버튼 제공
- 선택된 슬롯을 다시 클릭하면 1개씩 이동
- 수량 제한 적용

### Animal

- 동물 AI 상태머신 구현
- 초식동물(Passive Animal) / 육식동물(Hostile Animal) 구분
- 기본 상태: Idle, Roam
- 초식동물: 공격받으면 3초 동안 플레이어로부터 도주
- 육식동물: 공격받으면 플레이어를 추적·공격 / 플레이어 사망 시 Idle 복귀
- 공간 분할로 플레이어 주변 동물만 활성화

### Map 및 자원

- 타일 기반 맵 생성으로 변경
- 생성 시마다 Seed를 부여해 물·Ground·GrassGround 배치가 매번 달라짐
- 자원은 Ground 타일에만 생성
- 자원도 공간 분할로 플레이어 주변 자원만 활성화
- 물 타일은 높은 Collider로 이동 차단

### 낮/밤 시스템 & 리스폰

- 밤 안에 오두막에 입장하지 못하면 사망 처리
- 사망 시: 체력 50% 회복, 인벤토리 아이템 전부 소실 (동물에게 사망 시 동일)
- 오두막 입장 시: 체력 100% 회복, 인벤토리 유지

### 모바일 조이스틱 & 버튼

- Floating 조이스틱 추가 (왼쪽 클릭 시 생성)
- 인벤토리 버튼 및 설정 버튼 추가
- 설정 버튼: 현재 게임 일시정지 및 메인 메뉴 복귀로 임시 대체

### Upgrade System

- 창고, 인벤토리, 제작대 등 업그레이드 가능
- 화면 기준 오른쪽 대각선 아래(제단 아래) 파란색 포탈
- 상호작용 시 Upgrade Panel 표시
- 단계별로 요구 아이템이 달라짐
- 업그레이드 효과는 미구현 (예: 창고 용량 증가, 제작대 아이템 확장 등)

### 장비 착용 시스템 & 인벤토리

- 인벤토리 수량 제한 추가
- 용량 초과 시 1초짜리 팝업 표시
- 용량 초과 상태에서 아이템 획득 시 아이템 소실되지 않는 처리 미구현
- 플레이어 장비 착용 기능 구현
- 인벤토리에 버리기 버튼 추가 (수량 조절 후 드롭 가능)
- 아이템 클릭 시 정보 창에 Equip 버튼 추가
- 장비 효과는 미구현

## 플레이 방법

### 입력

| 입력          | 동작                                     |
| ------------- | ---------------------------------------- |
| 조이스틱      | 이동 (왼쪽 클릭 시 생성)                 |
| 액션 버튼     | 팔 휘두르기 (나무·돌·동물에 데미지)      |
| 상호작용 버튼 | 상호작용 가능한 오브젝트 근처에서 활성화 |
| 인벤토리 버튼 | 액션 버튼 위 가방 아이콘                 |
| 설정 버튼     | 메인 화면 복귀 (임시 구현)               |

### 게임 루프

7일마다 제단에 지정된 아이템을 제출해야 한다.

1. 자원(나무, 돌)을 채집한다.
2. 제작대에서 아이템을 제작하고 업그레이드한다.
3. 제단에서 제출 목록을 확인하고 아이템을 납부한다.
4. 밤이 되기 전에 오두막에 복귀한다.
5. 복귀 실패 또는 동물에게 사망 시 다음날로 넘어가며 인벤토리가 초기화된다.
   - 장착 중인 장비는 유지된다.

### 7일차 평가 흐름

1. 7일째 아침 도달 시 `TributeEvent.Evaluate()` 자동 호출
2. 7일째 아침 전까지 공물 납부를 완료해야 충족 처리
3. 충족 → 8일째 시작 / 미달 → 게임오버 → 타이틀 복귀
