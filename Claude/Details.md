# 🌲 부족 마을 생존 게임 - 세부 기획서

---

## 1. 게임 개요

- **배경**: 신비로운 제사 부족이 사는 마을 외곽 숲
- **설정**: 플레이어는 부족의 영역에 들어온 외지인. 부족의 너그러움 덕에 마을 외곽 오두막에서 머무는 것을 허락받았으나, 그 대가로 7일마다 부족이 모시는 신에게 바칠 **공물(供物)** 을 책임져야 한다
- **목표**: 7일마다 돌아오는 제사에 공물을 바쳐 부족의 인정을 받고 살아남는다
- **하루 플레이 시간**: 기본 3분 (실제 플레이 테스트 후 조정 예정)
  - 낮 활동: 약 2분 30초 (수집/제작/사냥)
  - 밤 연출: 약 30초 (취침 → 다음 날 전환)

---

## 2. 조작 방식

- **왼쪽 조이스틱**: 캐릭터 이동
- **오른쪽 버튼 1개**: 상호작용 (단일 버튼으로 모든 행동)
  - 버튼을 누르면 손/도구를 한 번 휘두름
  - 캐릭터 주변 일정 범위 내의 대상과 자동 상호작용
  - 예: 나무 옆에서 누르면 → 나무 채집 / 동물 옆에서 누르면 → 공격 / 아이템 위에서 누르면 → 줍기
- **장비**: 손에 든 장비에 따라 같은 버튼이 다른 효과를 냄
  - 맨손 → 채집량 적음 / 돌도끼 → 나무 채집 / 돌창 → 전투 및 채집

## 3. 수집 가능 물품

### 🌳 자연 자원 (지형 채집)

| 아이템 | 획득 방법 | 용도 |
|--------|-----------|------|
| 🪵 나무 (Wood) | 나무에서 버튼 액션 (도끼 사용 시 효율 증가) | 제작 기본 재료, 공물 |
| 🌰 나뭇가지 (Branch) | 바닥에서 줍기 or 나무 벌채 시 획득 | 도구 손잡이, 모닥불, 공물 |
| 🪨 돌 (Stone) | 바위에서 버튼 액션 (곡괭이 사용 시 효율 증가) | 제작 기본 재료, 공물 |
| 🌿 풀/잎사귀 (Grass) | 풀밭에서 버튼 액션 | 밧줄, 침구, 공물 |
| 🟫 점토 (Clay) | 강가/물웅덩이 근처 채집 | 제단 장식, 공물 |

### 🍴 식량 자원 (자연 채집)

| 아이템 | 획득 방법 | 용도 |
|--------|-----------|------|
| 🍄 버섯 (Mushroom) | 나무 주변 채집 | 식량, 배고픔 회복 (조리 시 효율 증가) |
| 🫐 열매 (Berries) | 덤불에서 채집 | 식량, 배고픔 소량 회복 |

### 🐾 동물 드롭 (사냥)

| 아이템 | 획득 방법 | 용도 |
|--------|-----------|------|
| 🐇 고기 (Meat) | 동물 처치 후 획득 | 식량 (조리 필요), 공물 |
| 🐻 가죽 (Hide) | 동물 처치 후 획득 | 방어구 제작, 공물 |
| 🦴 뼈 (Bone) | 중형/대형 동물 처치 후 획득 | 고급 도구 재료, 공물 (후반 필수) |
| 🪶 깃털 (Feather) | 새 사냥 시 획득 | 침대/가벼운 옷 제작, 공물 |

| 장비 | 재료 | 효과 |

| 뼈 창 (Bone Spear) | 나뭇가지 2 + 뼈 2 | 강화 무기, 곰 사냥 권장 |

| 장비 | 재료 | 효과 |
|------|------|------|

| 가죽 장갑 (Leather Gloves) | 가죽 2 | 채집 속도 소폭 증가 |

### 🏕️ 시설류

| 시설 | 재료 | 효과 |
|------|------|------|

| 침대 (Bed) | 나무 4 + 풀 4 | 밤 취침 → 다음 날 전환 |
| 조리대 (Cooking Stone) | 돌 5 | 버섯 요리 등 음식 조합 |
| 텐트 (Tent) | 나무 3 + 풀 5 | 야외 임시 거점 |

## 5. 동물 시스템

### 🐇 소형 동물 (즉사)

| 동물 | 처치 방식 | 드롭 |
|------|-----------|------|
| 토끼 | 버튼 1회 액션 or 덫 설치 | 고기 1, 가죽 1 |
| 다람쥐 | 버튼 1회 액션 | 열매 1~2 (특이 드롭) |

### 🦌 중형 동물 (짧은 전투)

| 동물 | 필요 타격 | 행동 패턴 | 드롭 |
|------|-----------|-----------|------|
| 사슴 | 3회 | 공격 없음, 도망감 | 고기 2, 가죽 2 |
| 여우 | 2회 | 물기 반격, 빠름 | 고기 1, 가죽 2 |

### 🐺🐻 대형 동물 (전투 필요)

| 동물 | 필요 타격 | 행동 패턴 | 드롭 | 등장 일자 |
|------|-----------|-----------|------|-----------|
| 늑대 | 5회 | 플레이어 추적, 물기 공격 | 고기 2, 가죽 3, 뼈 1 | 2일차~ |
| 곰 | 8회 | 느리지만 강한 타격 | 고기 3, 가죽 5, 뼈 2 | 4일차~ |

> 💡 전투는 단순하게 — 버튼 연타 + 타이밍 맞춰 이동으로 피하는 방식

## 6. 식량 / 배고픔 시스템

### 기본 개념
- 하루마다 **배고픔 게이지(Hunger Gauge)** 가 존재하며, 낮 동안 음식을 섭취해서 게이지를 채워야 함
- 취침 전까지 게이지를 **일정 수치 이상** 채우지 못하면 → 다음 날 체력 감소 상태로 시작
- **체력 회복 아이템은 없음** — 체력 회복은 오직 오두막에서 취침을 통해서만 가능

### 취침 및 체력 회복
- 밤에 **오두막으로 돌아와 취침**하면 → 다음 날 체력 **완전 회복**
- 오두막은 게임 시작 시 기본 제공됨 (침대/모닥불은 취침 필수 요소가 아님)
- 배고픔 게이지를 충분히 채운 상태로 취침 시 → 완전 회복
- 배고픔 게이지 미달 상태로 취침 시 → 다음 날 체력 일부 감소 후 시작

### 식량 목록

| 음식 | 조리 필요 | 배고픔 게이지 |
|------|-----------|--------------|
| 날고기 (Raw Meat) | ❌ (먹을 수는 있음) | 소량 회복 (날것 패널티로 효율 낮음) |
| 구운 고기 (Cooked Meat) | ✅ 모닥불 필요 | 다량 회복 |
| 열매 (Berries) | ❌ | 소량 회복 |
| 버섯 (Mushroom) | ❌ or ✅ | 중간 회복, 조리 시 효율 증가 |

## 7. 낮/밤 사이클

### ☀️ 낮 (메인 플레이, 약 2분 30초)
- 이동, 수집, 제작, 사냥, 탐험 등 모든 활동

### 🌙 밤 (연출 전환, 약 30초)
- 오두막으로 돌아오면 취침 가능 (침대/모닥불 별도 필요 없음)
- 취침 → 페이드 아웃 → 별하늘/바람 소리 연출 → 페이드 인 → 다음 날 아침
- 배고픔 게이지를 충분히 채운 경우 → 다음 날 체력 완전 회복
- 배고픔 게이지 미달 시 → 다음 날 체력 감소 상태로 시작
- 밤 중 별도 활동 없음

## 8. 제사 시스템 — 7일 사이클

### 기본 방향
- 부족은 7일에 한 번씩 신께 제사를 올리며, 외지인인 플레이어가 공물을 책임진다
- 게임 시작 시 **7일 뒤의 제사에서 어떤 신을 모실지, 어떤 공물이 필요한지 미리 예고**됨
- 매 사이클마다 **모시는 신과 요구 공물 조합이 무작위로 결정**되어 플레이가 매번 달라진다
- 플레이어는 7일 동안 재료를 모아 제단에 공물을 바치고, **7일 밤 → 다음 날 전환 연출 중 제사 의식이 진행**됨
- 제사를 무사히 넘기면 → 다음 7일 뒤 제사가 새로 예고되며 사이클이 반복
- 직접 조작 최소화, 텍스트 + 비주얼 연출로 진행

### 공물 제출 방식
- 마을 입구에 부족이 관리하는 **제단(Altar)** 이 배치되어 있음
- 플레이어가 제단에 접근해 상호작용 버튼을 누르면 → **공물 제출 창**이 열림
- 창에는 이번 제사에서 요구되는 공물 목록(랜덤 조합)과 현재 보유량이 표시됨
- 보유한 재료를 직접 제출하며, 한 번 제출한 재료는 회수 불가
- 모든 요구 공물을 채우면 → 제사 대비 완료 처리
- 일반 제작/시설 설치는 게임 내에 존재하지만, **제사 대비와는 무관** (생존 보조용)

### 사이클 흐름

```
[Day 1] 게임 시작 → 첫 번째 제사 예고 (모실 신 + 요구 공물 랜덤 결정)
   ↓ Day 1~7: 준비 기간 (수집, 제작, 공물 제출)
[Day 7 밤 → Day 8 전환 연출 중] 첫 번째 제사 의식 진행
   ↓ 공물 충족 → [Day 8] 두 번째 제사 예고 (새 신 + 새 공물 조합)
   ↓ Day 8~14: 다음 준비 기간
[Day 14 밤 → Day 15 전환 연출 중] 두 번째 제사 의식 진행
   ↓ 공물 충족 → [Day 15] 세 번째 제사 예고
   ↓ ... 반복 (7, 14, 21 밤 전환 시 제사)
```

### 모시는 신 (예시 풀)

매 사이클마다 아래 풀에서 신이 하나 선택되며, 신의 성격에 따라 공물 후보군 가중치가 달라진다. 최종 요구 공물은 가중치 기반으로 **랜덤 조합**되어 결정됨.

| 신 | 성격 | 선호 공물 후보 |
|----|------|---------------|
| 🌳 숲의 정령 | 자연 친화 | 나무, 나뭇가지, 풀, 점토 |
| 🏹 사냥의 신 | 야성/피의 신 | 고기, 가죽, 깃털, 뼈 |
| ⚒️ 대지의 신 | 강인함의 상징 | 돌, 점토, 뼈, 광물 |
| 🌙 달의 신 | 비밀스러운 신 | 깃털, 버섯, 가죽, 풀 |
| 🔥 불의 신 | 정화의 신 | 나뭇가지, 나무, 마른 풀, 뼈 |

### 공물 조합 규칙 (랜덤)
- 매 사이클마다 신을 1명 선택 → 신의 선호 풀에서 2~4종의 재료를 무작위 선정
- 각 재료의 요구량은 사이클 진행도에 따라 증가 (예: 1주차 베이스 → 2주차 ×1.5 → 3주차 ×2.0)
- 후반 사이클에는 뼈·광물 같은 희귀 재료가 포함될 확률 증가
- 사이클 시작 시점에 부족 장로의 예고 메시지로 신과 공물 목록이 동시에 공개

### 실패 조건 — 제물 의식
- 제사 시각까지 공물이 부족하면 → 부족이 플레이어에게 분노함
- **연출**: 화면이 어두워지고 부족 전사들이 횃불을 들고 다가옴 → 제단으로 끌려가는 플레이어 → 화면 페이드 아웃 → "당신은 신께 바쳐졌다" 텍스트
- 즉시 게임 오버 (체력 시스템과 무관하게 강제 종료)
- 게임 오버 시 모셨던 신에 맞춘 엔딩 텍스트 출력 (예: 숲의 정령 → "당신의 몸은 숲에 묻혔다", 불의 신 → "당신의 재가 하늘로 올랐다")

### 일반 실패 조건
- 낮/밤 중 체력이 0이 되면 일반 게임 오버 (제물 연출 없음)

## 9. 하루 일과 예시 (Day 3 기준)

```
[아침 시작] Day 3 — "사냥의 신께 가죽 8, 고기 6, 뼈 2를 바쳐야 한다" 예고 중
→ 오두막에서 출발 (완전한 체력으로 시작)
→ 사냥 루트 출발 (사슴/토끼 추적)
→ 덫 확인 → 토끼 1마리 포획 (가죽 1, 고기 1)
→ 사슴 발견 → 추적 후 처치 (가죽 2, 고기 2)
→ 모닥불로 구운 고기 조리 → 일부 섭취 (배고픔 게이지 충족)
→ 남은 가죽/고기/뼈를 가지고 제단으로 이동 → 공물 제출
→ 오두막으로 귀환 → 취침 → 체력 완전 회복
[다음 날로 전환]
```
