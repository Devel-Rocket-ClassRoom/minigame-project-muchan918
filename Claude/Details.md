# 🌲 7Days — 세부 기획서

> **최신 갱신: 2026-05-21 — 탈출 컨셉 + 시설 업그레이드 시스템 반영**

---

## 1. 게임 개요

- **배경**: 신비로운 부족 마을 외곽의 숲
- **설정**: 플레이어는 부족의 영역에 잡혀 외곽 오두막에서 사는 외지인. 부족이 7일마다 신께 바치는 공물을 책임진다.
- **목표**: **신비로운 돌**을 제작해 부족 몰래 제단에 봉헌하고 굴레에서 벗어난다.
- **하루 플레이 시간**: 약 3분 (낮 ~2분 30초 + 밤 ~30초 전환 연출)
- **한 게임 길이**: 약 5~7사이클(35~49일) → 1~2시간 분량

---

## 2. 조작 방식

- **왼쪽 조이스틱**: 캐릭터 이동
- **오른쪽 액션 버튼 1개**: 모든 상호작용 자동 분기
  - 자원 근처 → 채집 (손/도구 휘두름)
  - 아이템 위 → 줍기
  - 동물 근처 → 공격(액션 1회 휘두름, 채집과 동일 인터랙션)
  - 시설 / 제단 / 오두막 근처 → 해당 인터랙션 UI 오픈
- **장비**: 손에 든 도구에 따라 같은 버튼이 다른 효율
  - 맨손 < 돌도끼/돌곡괭이 < 강화 도구

> 전투는 의도적으로 최소화 — 별도 전투 시스템 없이 액션 버튼 1회 휘두름만.

---

## 3. 공간 구조 — 오두막 영역 vs 야생 영역

### 🏠 오두막 영역 (영구 / 안전)
- 오두막 + 주변 일정 반경
- 설치된 시설 / 저장소 누적이 **영구 유지**
- 자원 거의 없음
- 시각 경계: 울타리 또는 표지물 (TBD)

### 🌲 야생 영역 (사이클마다 재생성)
- 그 외 맵 전체
- 사이클 종료 시(7일차 밤 → 8일차 아침) **자원 리스폰**
- 멀리 갈수록 희귀 자원 확률 ↑
- 사이클이 진행될수록 평균 공물 요구량이 증가 → 더 멀리 가야 함

> 재생성 방식: **자원만 리스폰** 또는 **구역 고정 + 자원 위치 셔플** 중 결정 (구현 시 선택)

---

## 4. 시설 시스템

오두막 영역에는 게임 시작 시 **빈 표지판(자리만 있는 와이어프레임)**으로 시설 위치가 표시되어 있다. 플레이어가 표지판에 자원을 가져가 액션 버튼을 누르면 건설 → T1 시설 출현. 이후 같은 자리에서 자원 추가 투입으로 T2 → T3 업그레이드.

### 4-1. 작업대 (Workbench) — 도구·핵심 아이템 제작

| 단계 | 명칭 | 제작 가능 |
|------|------|-----------|
| T1 | 작업대 | 돌도끼, 돌곡괭이 (채집 효율 ↑) |
| T2 | 대장간 | 강화 도구 (채집 속도/수량 ↑↑) |
| T3 | 현자의 작업대 | **신비로운 돌** (단 1개 — 진엔딩 트리거) |

### 4-2. 가마솥 (Cauldron) — 요리·버프

| 단계 | 명칭 | 요리 |
|------|------|------|
| T1 | 모닥불 | 구운 고기 (배고픔 회복) |
| T2 | 가마솥 | 스튜·약초차 (배고픔 회복 ↑ + 일시 체력 회복) |
| T3 | 마법 가마솥 | 버프 요리 (이동 속도 ↑ / 채집량 ↑ / 시야 ↑ 등 — 후반 위험 지역 탐험 보조) |

> 요리 종류와 효과 수치는 TBD — 플레이테스트로 조정.

### 4-3. 저장소 (Storage) — 자원 누적

| 단계 | 명칭 | 용량 |
|------|------|------|
| T1 | 나무 상자 | 8슬롯 |
| T2 | 강화 상자 | 16슬롯 |
| T3 | 거대 창고 | 32슬롯 (또는 자원 종류별 무제한) |

> 인벤토리가 가득 차면 채집이 막히므로 1순위 건설 대상이 될 가능성 높음.

### 4-4. 제단 (Altar) — 업그레이드 없음

- 정적 시설 — 공물 제출용
- 7일차에 평가 기준
- **신비로운 돌 보유 시 UI 자동 전환** (5절 참조)

### 4-5. 건설 / 업그레이드 비용

- T1: 일반 자원만으로 건설 가능 (예: 나무 5 + 돌 3)
- T2: 일반 자원 다수 + 일부 중급 자원 (가죽·뼈 등)
- T3: **희귀 자원 필수** (위험 지역에서만 채집)

> 정확한 비용표는 플레이테스트로 결정.

## 5. 신비로운 돌 & 탈출 엔딩

### 5-1. 신비로운 돌이란?

- 작업대 T3(현자의 작업대)에서만 제작 가능한 **단 하나의 탈출 매개체**
- 제작 비용: 희귀 자원 다수 (정확한 조합은 TBD — 예: 희귀 광물 5 + 마른 뼈 3 + 일반 자원 다수)
- 제작 후 인벤토리에 보관

### 5-2. 제단 UI 분기

플레이어가 제단에 상호작용할 때:

| 인벤토리에 신비로운 돌 | 제단 UI 모드 | 가능 행동 |
|----------------------|-------------|-----------|
| 없음 | **공물 제출 모드** | 요구 공물 목록 표시 / 부분 제출 |
| 있음 | **봉헌 모드 (자동 전환)** | "신비로운 돌 봉헌" 버튼 — 누르면 즉시 엔딩 연출 |

> 봉헌 모드에서는 일반 공물 제출은 불가. 신비로운 돌이 공물을 대체하는 셈.

### 5-3. 탈출 엔딩 연출 (최소 버전)

1. 봉헌 확인 창 → "정말 봉헌하시겠습니까?"
2. 화면 페이드 아웃 + 사운드
3. 제단에서 빛이 솟구치는 파티클 연출
4. 페이드 인 → 광원 효과 + 텍스트: **"당신은 부족의 굴레에서 풀려났다"**
5. 메인 메뉴 / 다시 시작 버튼

> 부족 NPC 반응 연출은 2주차 후반/3주차 여유 시 추가.

### 5-4. 게임 시작 시 목표 안내

- 게임 시작 시 한 번 짧은 안내 텍스트 표시 (예: "부족의 굴레에서 벗어나는 길은 신비로운 돌뿐이다")
- 그 외 추가 스토리·서사는 최소화

## 6. 공물 시스템 (7일차 평가)

### 6-1. 1주차 최소 버전 (현재 작업 중)

| 항목 | 1주차 최소 사양 |
|------|----------------|
| 공물 조합 | **고정 1세트** (예: 나무 5, 돌 3, 풀 2) |
| 공물 예고 | 게임 시작 시 / 사이클 시작 시 텍스트로 1회 표시 |
| 제출 방식 | 제단 상호작용 → 제출 창 → 부분 제출 가능 |
| 7일차 트리거 | DayManager가 Day==7 도달 시 평가 |
| 충족 결과 | 다음 사이클 진행 (야생 재생성) |
| 미달 결과 | 제물 엔딩 (게임 오버) |

### 6-2. 2주차 이후 확장

- 사이클이 진행될수록 공물 요구량 증가 (소프트 한계)
- 신 풀 도입 가능성 (사이클마다 다른 신/공물 조합) — 콘텐츠 여유 시
- 공물 카테고리 다양화 (식물 / 광물 / 동물 산물)

## 7. 수집 가능 물품

### 🌳 자연 자원 (지형 채집)

| 아이템 | 획득 방법 | 용도 |
|--------|-----------|------|
| 🪵 나무 | 나무 채집 (도끼 효율 ↑) | 시설 건설 / 공물 |
| 🌰 나뭇가지 | 바닥 줍기 / 벌채 | 도구 손잡이 / 공물 |
| 🪨 돌 | 바위 채집 (곡괭이 효율 ↑) | 시설 건설 / 공물 |
| 🌿 풀 | 풀밭 채집 | 밧줄 / 공물 |
| 🟫 점토 | 강가 채집 | 시설 장식 / 공물 |

### 🍴 식량 자원

| 아이템 | 획득 방법 | 용도 |
|--------|-----------|------|
| 🍄 버섯 | 나무 주변 채집 | 식량 (가마솥 T2+ 조리 시 효율 ↑) |
| 🫐 열매 | 덤불 채집 | 식량 (소량 회복) |

### 🐾 (선택) 동물 드롭 — 미니 버전

| 동물 | 획득 방법 | 드롭 |
|------|-----------|------|
| 🐇 토끼 | 액션 1회 (즉사) | 고기 1, 가죽 1 |

> 사슴·여우·늑대·곰 등 중대형 동물은 콘텐츠 여유 시 추가.

### 💎 희귀 자원 (먼 지역)

| 아이템 | 획득 위치 | 용도 |
|--------|-----------|------|
| 희귀 광물 | 변두리 광맥 | 시설 T3 / 신비로운 돌 |
| 마른 뼈 | 변두리 유골/덫 | 시설 T2~T3 / 신비로운 돌 |
| 신성한 가죽 | 동물(미니 버전 추가 시) | 시설 T3 (선택) |

## 8. 식량 / 배고픔 시스템

### 기본 개념
- 하루마다 **배고픔 게이지**가 존재 (2주차에 본격 구현)
- 취침 전 일정 수치 이상 못 채우면 → 다음 날 체력 감소 상태로 시작
- 체력 회복은 오두막 취침으로만 가능

### 가마솥 단계별 식량
- T1 모닥불 → 구운 고기 (기본 배고픔 회복)
- T2 가마솥 → 스튜 (회복량 ↑ + 일시 체력 회복)
- T3 마법 가마솥 → 버프 요리 (능력치 강화)

## 9. 낮/밤 사이클

### ☀️ 낮 (메인 플레이, 약 2분 30초)
- 이동, 채집, 건설, 시설 이용, (선택) 사냥, 제단 인터랙션

### 🌙 밤 (전환 연출, 약 30초)
- 오두막 근처 취침 → 페이드 아웃 → 별하늘 연출 → 페이드 인 → 다음 날
- 배고픔 충족 시 체력 완전 회복 / 미달 시 패널티
- 밤 중 별도 활동 없음

### 7일째 밤 → 8일째 아침 (특수 전환)
- 평가 발동: 공물 충족 / 미달 판정
- 충족 → 야생 자원 리스폰 + 다음 사이클 시작
- 미달 → 제물 엔딩 연출
