# 7Days — 빌드 3 계획

---

## 이월 항목 (빌드 2 미완료)

- 업그레이드 효과 구현 (창고 용량 증가, 제작대 아이템 확장 등)
- 장비 착용 효과 구현 (체력 증가, 공격력 증가 등)
- 배고픔 시스템
- 요리 시스템 (가마솥)
- 제단 난이도 증가 (사이클마다 공물 요구량 증가)
- UI 스프라이트 및 폰트 할당
- 용량 초과 시 아이템 소실 방지 처리

---

## 구현 목표

### 1. 가마솥 요리 & 배고픔 시스템

- 제작대와 동일한 로직으로 요리 시스템 도입
- 음식 섭취 시 배고픔 게이지 회복
- 배고픔 시스템 메커니즘 설계

### 2. 업그레이드 효과 구현

- 창고 용량 증가
- 인벤토리 용량 증가
- 제작대 아이템 확장
- 동물 스폰 종류 증가
- 자원 스폰 종류 증가

### 3. 세이브 & 로드 시스템

- 수동 저장 버튼
- Auto Save 기능
- 이어하기 / 새로하기 선택
- 암호화 적용

### 4. UI 에셋 적용

- 각종 Panel 스프라이트
- World Item 아이콘
- 기타 UI 요소

### 5. 오두막 Indicator

- 화면 중앙에 Indicator Panel 생성 (앵커 MinX: 0.2, MinY: 0.2, MaxX: 0.8, MaxY: 0.8 검토 중)
- 오두막이 화면 밖으로 벗어나면 Indicator 활성화
- 오두막이 화면 안으로 복귀하면 Indicator 비활성화

### 6. 튜토리얼

### 7. 사운드 추가

### 8. 이펙트 추가

### 9. 콘텐츠 확장

- 다양한 아이템
- 다양한 동물
- 다양한 장비
- 다양한 자원
- 건물 오브젝트 배치
- 맵 끝 처리

### 10. 연출

- 낮/밤 전환 연출
- 7일차 판단 연출
