# 7Days — 빌드 2 계획

> 다음 빌드 목표일: 2026-05-29 (Day 14)

---

## 구현 목표

### 1. 공물 제출 UI

- 공물 요구 데이터 구조 정의 (예: 나무 5, 돌 3 고정 1세트)
- 제단 패널에 요구 목록 / 보유량 표시
- 부분 제출 가능 (누적 제출량 표시)
- 제출 시 PlayerInventory에서 해당 아이템 차감
- 충족 완료 시 `TributeEvent.IsTributeFulfilled = true` 세팅

### 2. 7일차 결과 화면 UI

- 충족(통과) 결과: "다음 사이클로 이어집니다" + 확인 버튼
- 미달(제물) 결과: "제물로 바쳐졌다" + 타이틀 복귀 버튼
- `DayNightCycle.SetMorning()` 분기에서 각각 화면 활성화

### 3. 동물 AI

- 동물 베이스 클래스 작성 후 초식 / 육식 동물 구현
- 초식 동물: 피해를 입으면 도망감
- 육식 동물: 피해를 입으면 반격함

### 4. 아이템 제작 시스템

- 작업대에서 상호작용 버튼을 누르면 제작 창 표시
- 현재 인벤토리 보유 아이템 기준으로 제작 가능 목록 표시

### 5. 아이템 저장 시스템

- 창고에서 상호작용 버튼을 누르면 저장 창 표시
- 인벤토리 ↔ 창고 간 아이템 이동 가능
- 아이템 클릭 시 수량 선택 창 표시

### 6. 요리 시스템

- 아이템 제작 시스템과 동일한 구조로 요리 제작
- 음식 아이템 사용(섭취) 기능
