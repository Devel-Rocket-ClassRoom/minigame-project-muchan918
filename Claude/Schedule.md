# 7Days — 개발 일정표

> 개발 기간: 3주 (2026.05.17 ~ 2026.06.06)
> 플랫폼: Android (Unity 6.3, URP)
> 목표: Play Store 출시
> **최신 갱신: 2026-05-21 — 탈출 컨셉 + 시설 업그레이드 시스템 반영**

---

## 📌 핵심 디자인 요약

- **단일 진엔딩**: 신비로운 돌 → 제단 봉헌 → 탈출
- **실패 엔딩**: 7일차 공물 미달 → 제물
- **시설 3종**: 작업대(T1~T3) / 가마솥(T1~T3) / 저장소(T1~T3)
- **제단**: 정적, 신비로운 돌 보유 시 UI 자동 봉헌 모드 전환
- **공간 이원화**: 오두막 영역(영구) + 야생(사이클마다 자원 재생성)
