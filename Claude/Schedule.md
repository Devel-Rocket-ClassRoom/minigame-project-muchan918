# 7Days — 개발 일정표

> 개발 기간: 3주 (2026.05.17 ~ 2026.06.06)  
> 플랫폼: Android (Unity 6.3, URP)  
> 목표: Play Store 출시

---

## 1주차 (05.17 ~ 05.23) — 핵심 시스템 구현

게임 전체가 돌아가는 뼈대를 완성하는 주간.  
이 주차가 끝나면 "채집 → 취침 → 다음 날" 기본 루프가 작동해야 한다.

### Day 1 (05.17)
- [ ] 프로젝트 세팅 (URP, Input System, NavMesh 확인)
- [ ] 기본 씬 구성 (맵 레이아웃, 오두막 배치)
- [ ] 플레이어 이동 (조이스틱 입력 → 캐릭터 이동)

### Day 2 (05.18)
- [ ] 카메라 탑다운 추적 구현
- [ ] 낮/밤 시간 시스템 (타이머, 낮↔밤 전환, 날짜 카운트)
- [ ] 취침 시스템 (오두막 근처 밤 → 페이드 아웃/인 → 다음 날 전환)

### Day 3 (05.19)
- [ ] 채집 가능 오브젝트 구조 설계 (ScriptableObject 기반 자원 데이터)
- [ ] 채집 시스템 (나무, 돌, 풀, 열매 — 액션 버튼 1회 상호작용)
- [ ] 바닥 아이템 드롭 및 줍기

### Day 4 (05.20)
- [ ] 아이템 & 인벤토리 시스템 (슬롯 구조, 아이템 추가/제거)
- [ ] 체력(HP) 시스템 기초
- [ ] 배고픔(Hunger) 게이지 시스템 기초 (취침 시 게이지 체크 → 체력 패널티)

### Day 5 (05.21)
- [ ] 기본 동물 AI — 소형 동물 (토끼: 배회 → 도망, 즉사)
- [ ] 기본 동물 AI — 중형 동물 (사슴: 배회 → 도망, 3회 처치)
- [ ] 처치 시 아이템 드롭 (고기, 가죽)

### Day 6 (05.22)
- [ ] 액션 버튼 상호작용 통합 (채집 / 공격 / 줍기 자동 판별)
- [ ] 모닥불 설치 및 고기 조리 기능 (날고기 → 구운 고기)
- [ ] 식량 섭취 → 배고픔 게이지 회복

### Day 7 (05.23)
- [ ] 1주차 통합 테스트 (채집 → 조리 → 취침 → 다음 날 루프 확인)
- [ ] 발견된 버그 수정
- [ ] 기본 루프 플레이 가능 여부 확인

---

## 2주차 (05.24 ~ 05.30) — 게임 시스템 정교화

아이템 요소, 이벤트 시스템, UI를 추가하여 실제 게임다운 완성도를 높이는 주간.  
이 주차가 끝나면 제사 이벤트까지 포함한 전체 게임 루프가 완성되어야 한다.

### Day 8 (05.24)
- [ ] 제작(Crafting) 시스템 구현 (재료 소모 → 아이템 생성)
- [ ] 제작 레시피 데이터 설계 (ScriptableObject)
- [ ] 도구류 제작 (돌도끼, 돌곡괭이, 덫)

### Day 9 (05.25)
- [ ] 무기류 제작 (돌창, 돌칼, 뼈 창)
- [ ] 방어구류 제작 (가죽 조끼, 가죽 장갑)
- [ ] 시설류 제작 (모닥불, 침대, 조리대, 텐트)
- [ ] 장비 장착/해제 시스템 (액션 버튼 효과 변경)

### Day 10 (05.26)
- [ ] 대형 동물 AI — 늑대 (추적 → 물기 공격, 5회 처치, 2일차~)
- [ ] 대형 동물 AI — 곰 (느린 추적 → 강한 타격, 8회 처치, 4일차~)
- [ ] 덫 설치 및 자동 포획 기능

### Day 11 (05.27)
- [ ] 제사 시스템 구조 설계 (ScriptableObject 기반 — 신 데이터, 공물 후보 풀)
- [ ] 제단 시스템 (상호작용 → 공물 제출 창 → 부분 제출)
- [ ] 제사 예고 공지 시스템 (사이클 시작 시 모실 신 + 공물 목록 표시)

### Day 12 (05.28)
- [ ] 공물 조합 랜덤 생성 로직 (신별 선호 풀 + 가중치 + 사이클 진행도 반영)
- [ ] 제사 의식 연출 (Day 7 / 14 / 21 밤 전환 중 발생)
- [ ] 공물 충족 시 사이클 진행 / 미달 시 제물 게임 오버 처리

### Day 13 (05.29)
- [ ] UI 구현 — HUD (체력 바, 배고픔 게이지, 날짜/시간 표시)
- [ ] UI 구현 — 조이스틱, 액션 버튼, 인벤토리 버튼
- [ ] UI 구현 — 인벤토리 & 장비 통합 창 (아이템 슬롯, 장비 슬롯, 제작 슬롯)
- [ ] UI 구현 — 오두막 방향 인디케이터

### Day 14 (05.30)
- [ ] UI 구현 — 제단 공물 제출 창, 제사 예고/의식/제물 연출 UI
- [ ] UI 구현 — 일시정지 메뉴 (재개, 설정, 메인 메뉴)
- [ ] 2주차 통합 테스트 (제사 이벤트 전체 루프 확인)

---

## 3주차 (05.31 ~ 06.06) — 테스트 및 버그 수정

전체 게임 품질을 높이고 출시 준비를 완료하는 주간.

### Day 15 (05.31)
- [ ] 치팅 툴 구현 (제사 강제 발생, 공물 자동 충족, 자원 조절, 날짜 변경 등 QA용 인게임 툴)
- [ ] 세이브 / 로드 시스템 (저장 단위: 게임 내 하루)

### Day 16 (06.01)
- [ ] 전체 게임 플레이테스트 1차
- [ ] 버그 리스트 정리 및 우선순위 분류

### Day 17 (06.02)
- [ ] 버그 수정 (1차 리스트)
- [ ] 밸런스 조정 (하루 플레이 시간, 배고픔/체력 수치, 채집량)

### Day 18 (06.03)
- [ ] 전체 게임 플레이테스트 2차
- [ ] 버그 수정 (2차 리스트)
- [ ] 사운드 효과(SFX) 및 BGM 적용

### Day 19 (06.04)
- [ ] 안드로이드 빌드 테스트 (실기기 확인)
- [ ] 성능 최적화 (드로우콜, 물리, NavMesh 등)
- [ ] UI 해상도 대응 확인

### Day 20 (06.05)
- [ ] Play Store 출시 준비 (앱 아이콘, 스크린샷, 설명 텍스트 등)
- [ ] 최종 빌드 생성 및 검토

### Day 21 (06.06)
- [ ] 최종 버그 수정 및 빌드 확정
- [ ] Play Store 제출
