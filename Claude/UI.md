# UI 기획서

> **최신 갱신: 2026-05-21 — 탈출 컨셉(신비로운 돌) + 시설 업그레이드 UI 반영**

---

## 0. 1주차 필수 UI (최소 동작용)

핵심 루프(채집 → 습득 → 7일차 → 생존 판정)를 한 번 완주시키기 위해 1주차에 반드시 구현되어야 하는 UI:

| UI | 상태 | 비고 |
|----|------|------|
| 조이스틱 (이동) | ✅ 구현 | — |
| 액션 버튼 (단일 상호작용) | ✅ 구현 | 채집/줍기/제단 상호작용 공통 |
| 인벤토리 버튼 + 슬롯 창 | ✅ 구현 | 메인 씬에서 동작 |
| HP 체력바 | ✅ 구현 | — |
| 날짜/시간 표시 (Day N) | 🟡 부분 | DateUI 존재, 표시 완성 필요 |
| **제단 공물 제출 창** | ❌ **1주차 신규** | 8절 참고 — 최소 버전(공물 제출 모드만) |
| **공물 예고 패널** | ❌ **1주차 신규** | 사이클 시작 시 1회 표시 (요구 공물 텍스트 위주) |
| **게임 시작 안내 텍스트** | ❌ **1주차 신규** | 6절 참고 — 목표 한 줄 ("신비로운 돌만이…") |
| **결과 화면** | ❌ **1주차 신규** | 생존(다음 사이클) / 제물(게임 오버) 텍스트 + 재시작 버튼 |

> **3주차 추가 예정 UI**: 봉헌 모드 제단 창(8-2절), 탈출 엔딩 연출(6절), 시설 건설/업그레이드 UI(9절).

배고픔 게이지, 일시정지, 오두막 방향 인디케이터, 풀스크린 연출 등은 2주차 이후로 이연.

## 1. 오두막 방향 인디케이터

- 미니맵 없이 **화면 가장자리에 오두막 방향을 고정 표시**하는 인디케이터 방식 사용
- 플레이어가 어디에 있든 항상 오두막 방향을 가리키는 아이콘이 화면 끝에 표시됨
- 오두막에 가까워질수록 인디케이터가 화면 중앙 방향으로 이동하다가 오두막이 시야에 들어오면 사라짐

---

## 2. 조이스틱 (이동)

- 화면 왼쪽 하단에 고정 표시
- 드래그 방향으로 캐릭터 이동
- 조이스틱에서 손을 떼면 캐릭터 정지

---

## 3. 액션 버튼

- 화면 오른쪽 하단에 고정 표시
- 버튼 위에 현재 장착 중인 장비 아이콘 표시, 장착 장비가 없으면 주먹 아이콘 표시
- **단일 버튼**으로 모든 상호작용 처리
- 버튼을 누르면 캐릭터 주변 일정 범위 내의 대상과 자동 상호작용
  - 채집 가능한 오브젝트 근처 → 채집 (손/도구 1회 휘두름)
  - 동물/몬스터 근처 → 공격
  - 아이템 위 → 줍기
  - 제단 근처 → 공물 제출 창 열기
  - 오두막 근처 (밤) → 취침

---

## 4. 인벤토리 버튼

- 화면 오른쪽 상단에 고정 표시
- 누르면 인벤토리 & 장비 통합 창이 열림
- 창이 열려있는 동안 조이스틱/액션 버튼 입력 비활성화
- 다시 누르거나 창 바깥을 탭하면 닫힘

### 인벤토리 & 장비 통합 창 구성

창은 좌/우 두 영역으로 나뉨

**왼쪽 — 아이템 슬롯**
- 보유 중인 아이템 목록을 슬롯 형식으로 표시 (아이템명, 수량)
- 장비 아이템 슬롯을 탭하면 오른쪽 장비 슬롯에 장착 / 이미 장착 중이면 해제

**오른쪽 — 장비 슬롯 & 캐릭터**
- 캐릭터 실루엣(또는 아이콘)이 중앙에 표시됨
- 캐릭터 주변에 장비 슬롯 배치
  - 무기 슬롯 1개
  - 방어구 슬롯 (가죽 조끼, 가죽 장갑 등 추후 추가 시 확장)
- 슬롯에 장착된 장비 아이콘 표시, 빈 슬롯이면 빈 칸으로 표시
- 장착된 장비 슬롯을 탭하면 해제

**하단 — 제작 슬롯**
- 제작 가능한 항목 목록을 슬롯 형식으로 표시
- 보유 재료가 충족된 항목은 활성화, 부족하면 비활성화(흐리게 표시)
- 활성화된 항목을 탭하면 확인 버튼 표시 → 확인 시 재료 소모 후 제작
- 제작 완료된 아이템은 자동으로 왼쪽 아이템 슬롯에 추가

## 5. HUD

항상 화면에 표시되는 요소들. 버튼이 없는 영역과 겹치지 않도록 배치.

- **체력 바 (HP)**: 현재 체력 / 최대 체력 표시
- **배고픔 게이지**: 현재 배고픔 수치 표시, 취침 전까지 일정 수치 이상 채워야 함
- **날짜 표시**: 현재 며칠째인지 표시 (예: Day 3), 다음 공물 평가까지 남은 일수도 함께 표시 (예: 공물까지 D-4)
- **시간 표시**: 현재 낮/밤 여부와 남은 시간 표시 (예: 타이머 또는 태양/달 아이콘)

## 6. 이벤트 연출 UI

> ⚠️ 일부 미정 — 추후 결정 예정

게임 흐름상 필요한 연출 UI 목록. 톤은 미니멀(텍스트 + 페이드 + 파티클) 우선, 풀스크린 컷씬은 여유 시.

### 6-1. 게임 시작 안내 (1주차 신규 / 최소 버전)
- 신규 게임 시작 시 1회 표시 (스킵 가능)
- 한두 줄 텍스트로 목표 안내
  - 예: **"부족의 굴레에서 벗어나는 길은 신비로운 돌뿐이다."**
- 추가 서사·NPC 등장은 최소화 (스토리 부담 ↓)

### 6-2. 공물 예고 (1주차 신규)
- 사이클 시작 시 1회 표시 (스킵 가능)
- 이번 사이클 요구 공물 목록을 텍스트로 안내
- 1주차에는 부족 장로 캐릭터·신 아이콘 등 없이 텍스트 위주 최소 UI
- 2주차 이후 신 풀 확장 시 신 아이콘 추가 가능

### 6-3. 취침 전환 연출 (1주차)
- 페이드 아웃 → 밤 사운드 → 다음 날 페이드 인
- 7일째 → 8일째 전환 시에는 공물 평가 결과 분기로 연결

### 6-4. 7일차 공물 평가 결과
- **충족(생존)**: 짧은 텍스트 ("부족이 당신을 살려둔다") → 다음 사이클 시작
- **미달(제물 엔딩)**: 화면 어두워짐 → "당신은 신께 바쳐졌다" 텍스트 → 메인 메뉴 / 재시작 버튼
- 1주차에는 텍스트 + 페이드만, 부족 전사 컷씬 등은 여유 시 추가

### 6-5. 탈출 엔딩 연출 (3주차 신규 — 진엔딩)
- 트리거: 제단에 **신비로운 돌 봉헌** (8-2절 참조)
- 봉헌 확인 창 → "정말 봉헌하시겠습니까?" (Y/N)
- 확인 시 연출 진행:
  1. 화면 페이드 아웃 + 사운드 (저음 → 상승)
  2. 제단에서 빛이 솟구치는 파티클 연출
  3. 페이드 인 → 광원 효과
  4. 텍스트: **"당신은 부족의 굴레에서 풀려났다"**
  5. 메인 메뉴 / 다시 시작 버튼
- 부족 NPC 반응 연출은 3주차 여유 시 추가

구체적인 비주얼 표현·SFX는 플레이테스트 후 결정.

## 7. 일시정지 버튼

- 화면 왼쪽 상단에 고정 표시
- 누르면 게임이 일시정지되고 일시정지 메뉴가 화면 중앙에 표시
- 일시정지 메뉴 구성:
  - **재개**: 메뉴를 닫고 게임 재개
  - **설정**: 사운드(BGM/SFX) 볼륨 조절 등 기본 설정 화면으로 이동
  - **메인 메뉴로**: 현재 진행 상황을 저장하고 메인 메뉴로 이동

## 8. 제단 공물 제출 창

제단은 **인벤토리의 신비로운 돌 보유 여부에 따라 UI 모드가 자동 전환**된다.

| 인벤토리에 신비로운 돌 | 제단 UI 모드 |
|----------------------|-------------|
| 없음 | **공물 제출 모드 (8-1)** |
| 있음 | **봉헌 모드 (8-2)** — 자동 전환, 공물 제출 불가 |

### 8-1. 공물 제출 모드 (기본 / 1주차)

- 마을 입구 제단에 액션 버튼으로 상호작용 시 열림
- 상단에 이번 사이클 요구 공물 정보 표시 (1주차: 신/아이콘 없이 텍스트만 / 2주차 이후 확장)
- 요구 공물 목록과 현재 보유량 / 누적 제출량 표시
- 각 재료 행에 **부분 제출 버튼** — 가진 만큼 즉시 제출 가능 (한 번 제출하면 회수 불가)
- 모든 요구 공물이 채워지면 상단에 "공물 충족" 표시 → 7일차 평가 시 통과
- 닫기 버튼 또는 창 바깥 탭으로 닫힘

### 8-2. 봉헌 모드 (3주차 신규 — 진엔딩 트리거)

- 인벤토리에 **신비로운 돌**이 있을 때 제단 상호작용 시 자동으로 이 모드로 열림
- 화면 중앙에 신비로운 돌 아이콘 + 간단한 설명 텍스트 (예: "신비로운 돌을 제단에 봉헌하면 굴레에서 풀려난다")
- 단일 버튼: **"봉헌하기"**
- 확인 창("정말 봉헌하시겠습니까?") → 확인 시 **탈출 엔딩 연출** 발동 (6-5절)
- 닫기 가능 — 아직 봉헌 시점이 아니라고 판단하면 그냥 닫고 나갈 수 있음
- 봉헌 모드에서는 일반 공물 제출 불가 (신비로운 돌을 인벤토리에서 빼면 다시 공물 제출 모드로 돌아옴)

> 디자인 의도: 신비로운 돌을 만든 시점부터는 제단의 의미가 "공물 제출처"에서 "탈출의 문"으로 자연 전환되며, 마지막 사이클의 평가는 봉헌 한 번으로 압축된다.

## 9. 시설 건설 / 업그레이드 UI (2주차 신규)

오두막 영역의 빈 표지판(작업대 / 가마솥 / 저장소 자리)에 액션 버튼으로 상호작용 시 열림.

### 9-1. 빈 표지판 → 건설 창
- 시설 이름 + 단계(T1) 표시
- 요구 자원 목록과 현재 보유량 표시 (부족 시 빨간색)
- **건설 버튼** — 자원 충족 시 활성화, 누르면 인벤토리/저장소에서 자원 차감 + T1 시설 출현

### 9-2. 기존 시설 → 업그레이드 창
- 현재 단계(T1 또는 T2) + 효과 요약 표시
- 다음 단계(T2 또는 T3) 요구 자원 목록 + 효과 미리보기
- **업그레이드 버튼** — 자원 충족 시 활성화

### 9-3. 작업대 / 가마솥 — 제작 메뉴 통합
- 시설 상호작용 시 업그레이드 외에 **제작 메뉴** 탭도 함께 표시
  - 작업대: 도구, (T3) 신비로운 돌
  - 가마솥: 단계별 요리
- 인벤토리 창 하단의 제작 슬롯과는 별개 — 시설 의존 제작은 시설 자체에서만 가능

> 인벤토리 하단 제작 슬롯(4절)은 시설 무관 기본 제작(예: 임시 도구·횃불 등)으로 한정할지, 폐지할지는 TBD.
