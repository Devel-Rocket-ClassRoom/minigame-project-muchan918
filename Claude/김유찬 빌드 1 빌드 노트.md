# 7Days — 1차 빌드 노트

> 빌드 날짜: 2026-05-22
> 빌드 대상: Android / Windows
> 개발 단계: 1주차 Day 7 (05.22) — 제단/공물 UI 작업 전 상태
> Unity 버전: 6.3 / URP

---

## 구현된 기능

### 플레이어

- **이동 & 회전**: InputSystem Move 액션 → Rigidbody XZ 이동, 이동 방향 Slerp 회전
- **애니메이터 연동**: Speed 파라미터로 이동/정지 블렌딩
- **체력(HP) 시스템**: maxHp 100, TakeDamage / Recover / Die 구현, UI 슬라이더 연동
- **인벤토리**: 슬롯 기반 아이템 추가/제거, UI 슬롯 목록 + 아이템 정보 패널
- **인터랙션 시스템**: IInteractable 인터페이스 기반 — 자원/오두막/제단 자동 분기

### 카메라 & 씬

- 탑다운 카메라 추적 + 오클루전 투명화 처리
- 화면비(Aspect) 자동 대응 (CameraAspectFit)
- SafeArea 대응 UI 패널 (SafeAreaPanel, SafeAreaCameraAdjuster)

### 낮/밤 & 날짜

- 1일 = **약 18초** (낮 11초 → 노을 3초 → 밤 4초)
- DirectionalLight 강도 Lerp로 노을 연출
- DateUI로 현재 날짜/시각 HUD 표시
- 취침 시스템: 오두막 근처 인터랙션 → SetMorning (다음 날 아침, 체력 회복)
- 오두막 Interaction은 파란색 원기둥으로 임시 처리

### 채집 & 자원

- ResourceData ScriptableObject 기반 자원 정의 (나무/돌/풀/열매 등)
- ResourceObject: 채집 가능한 오브젝트(나무, 돌) - 파괴 시 아이템 드랍
- WorldItem: 바닥 아이템 생성 & 줍기 (IDroppable 인터페이스)
- ItemData / ItemAsset ScriptableObject + DataTableManager 연동

### 7일차 평가 (TributeEvent)

- `DayNightCycle.SetMorning()`에서 `CurrentDay % 7 == 0` 조건으로 평가 분기
- 충족 시: `AssignNewEvent()` → 다음 사이클 진행
- 미달 시: `PlayerHealth.Die()` → 게임오버 UI 표시 → 타이틀 씬 복귀
- AssignNewEvent 함수는 미구현

### 제단 (Altar)

- AltarInteraction: 인터랙션 시 altarPanel 활성화 + 게임 일시정지
- "공물 납부" 버튼 → `IsTributeFulfilled = true` 세팅 (현재 임시 처리)
- 제단 Interaction은 빨간색 원기둥으로 임시 처리

### 기타

- MapGenerator / GroundChunkGenerator (맵 생성)
- GamePause: Pause / Resume (timeScale 제어)
- GameOverUI: 타이틀 씬으로 복귀 버튼

## 임시 처리된 부분

- 공물 요구사항: `IsTributeFulfilled` bool 플래그만 존재, 실제 요구 아이템/수량 없음
- 제단 공물 제출 UI: 버튼 1개 -> 무조건 충족 처리
- 7일차 결과 화면: 충족하면 다음날로 넘어가고 충족 못하면 게임 오버 UI 생성
- 낮,밤 연출: 현재 24시가 되어도 아무런 동작을 하지 않음(원래는 죽어서 오두막에서 부활)

## 플레이 방법

### 입력

- WASD 방향키: 캐릭터 이동
- 스페이스바: 액션 버튼(팔을 휘두르고 Collider 범위 안에 들어오면 나무, 돌에 데미지 준다.)
- 인터렉션 버튼: 상호작용 가능한 곳에 가면 Interaction 버튼이 나온다(드랍 아이템, 오두막, 제단)
- 오두막 인터렉션: 파란 원기둥에 가서 Interaction 버튼을 누르면 다음날로 넘어감
- 제단 인터렉션(빨간 원기둥): 공물 패널을 열 수 있다 -> "상납 완료" 버튼을 누르면 해당 7일차 조건 충족(임시처리)
- P키: 데미지 10 받음(개발용 테스트)
- Tab키: 인벤토리 창 열고 닫을 수 있다. (닫을때도 tab 눌러야함 x버튼 미구현)

### 루프

- WASD키로 나무나 돌 근처에 가서 스페이스바를 통해 채집할 수 있다.
- 떨어진 아이템 근처에서 Interaction 버튼을 눌러 인벤토리에 넣을 수 있음
- 오두막에 도착해서 Interaction을 버튼을 눌러 다음날로 넘어감
- 제단(빨간색 원기둥)에 가서 Interaction 버튼을 눌러 공물 패널을 열 수 있다.
- "상납 완료"를 눌러 IsTributeFulfill을 true로 만든다
- 오두막에서 Interaction을 누르다 보면 7일마다 IsTributeFulfill을 검사한다
- "상납 완료"를 했으면 다음날로 넘어가고 "상납 완료"를 못했으면 게임 오버

### 7일차 평가 흐름

1. 7일째 아침 도달 시 자동으로 `TributeEvent.Evaluate()` 호출
2. 제단에서 "공물 납부" 버튼을 **7일째 아침 전**에 눌러야 충족 처리됨
3. 충족 → 8일째 시작 / 미달 → 게임오버 화면 → 타이틀로 복귀

> ⚠️ 현재 공물 제출은 실제 인벤토리 차감 없이 버튼만 누르면 충족됩니다.
